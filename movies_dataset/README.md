## 📌 Overview

This repository contains a **movie dataset** collected using **web scraping techniques**.
The dataset includes information about thousands of movies such as their titles, descriptions, popularity, ratings, and release dates.

This dataset can be used for:

* 🎯 Machine Learning projects
* 🎥 Movie recommendation systems
* 📊 Data analysis and visualization
* 🧠 Natural Language Processing (NLP) on movie descriptions

---

## 📊 Dataset Information

| Feature          | Description                               |
| ---------------- | ----------------------------------------- |
| **id**           | Unique movie identifier                   |
| **title**        | Name of the movie                         |
| **overview**     | Short description or summary of the movie |
| **release_date** | Movie release date                        |
| **popularity**   | Popularity score of the movie             |
| **vote_average** | Average rating given by users             |
| **vote_count**   | Number of votes received                  |

📦 **Total Movies:** ~8550+
